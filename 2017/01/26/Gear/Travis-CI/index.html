添加环境变量

  • 方法一
    Travis CI->Setting添加GITHUB_API_KEY=, Display value in build log 设置为 Off.

  • 方法二
    进入仓库根目录下运行 travis encrypt -a GITHUB_API_KEY <token> GITHUB_API_KEY会自动加到.travis.yml
